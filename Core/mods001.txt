53. did not removed Dead time counters. i was trying to stay off long enough to guarantee load had discharged. but don't need that.
         or at least we are not worrying about that, since lights are careful not to apply to much capacitive load to a
         circuit. lights either use active PF correction in which case the load is inductive or resistive, or they
         use passive PF correction, which involves presenting a deliberately small capacitance to the mains leads
         but in any case you don't see that until mains voltage exceeds stored voltage since the capacitor is usually
         behind a bridge rectifier. if there is a true capacitive load - such as perhaps a PF correction bank of capacitors
         on an old iron core transformer fluorescent ballast, they we may be in trouble. But such ballasts are not
         dimmable, so we would only be on 100% duty anyway. Also such lights are purely capacitive until the light
         strikes at which time they start looking resistive. We could implement a system where if the light looks
         capacitive and then becomes resistive, disable dimming because the light is an old fluoro. at turn off of
         and old fluoro, the fluoro may present some voltage back into the mains until its PF correction capacitors
         are discharged. So we would like to disable immediate turn on again for a number of cycles when an old
         fluoro is detected to allow the PF capacitor to completely discharge. to tell the difference between a true
         capacitor load and a CFL or LED electronic light, you could filter current to the point where noise is eliminated
         but at least the 3rd harmonic is still present. then measure the peak, compensate for filtering,
         estimate what the rms should have been had the waveform been purely sinusoidal and if RMS is smaller
         than that, then you know you are dealing with a CFL or LED. Additionally, a CFL or LED does not shift
         the mains current leading too much because it is only on close to the middle of the 50Hz cycle. a true
         capacitive load does shift it. A typical 36W batten requires a 1.3H iron core transformer. This requires
         about 5.6uF to obtain unity PF. In the worst case, the fluorescent lamp does not strike at all and the
         PF correction capacitor is the only component in circuit. In this case that capacitor may be left with 340V stored on it.
         Typically 80V to 120V max can be left on the PF correction capacitor at turn off after around 5ms.

notes
=====
1. use "DMA transfers complete" interrupt at end of conversion of ADC4.
2. interrupts enables and priorities are in NVIC under System Core under Categories under Pinout & Congfiguration under lumira_hv.ioc
3. ISR names are first weakly defined in stm32c0xx_it.h
4. ISRs are in stm32c0xx_it.c
5. DMA ISR is enabled at the NVIC level in dma.c with NVIC_EnableIRQ(DMA1_Channel1_IRQn)
6. The type of DMA interrupt is enabled in main.c with LL_DMA_EnableIT_TC(DMA1, LL_DMA_CHANNEL_1) and LL_DMA_EnableIT_TE(DMA1, LL_DMA_CHANNEL_1) for
	transfer complete and transfer error. LL_DMA_EnableIT_HT(DMA1, LL_DMA_CHANNEL_1) is half transfer and is disabled.


.ioc setup
----------
ADC setup:
1. scan mode enabled
2. continuous mode disabled // generates multiple sequences of conversions or just one
3. DMA continuous requests disabled // after the first sequence is transferred, keep or dont keep transferring conversions
4. number of conversions 5

dma setup:
1. mode circular // so that the DMA resets ready for next 5 transfers upon a timer trigger
2. transfer complete interrupt enable

timer 3:
counter mode up
prescaler = xxxx
auto reload (ARR) = yyyy (defines trigger period)

Trigger 0 setup:
Master Mode Selection (TRGO) = Update Event

make ADC wait for timer 3:
External Trigger Source = TIM3 TRGO
Trigger Edge = Rising

LL
==
ADC, setup:
LL_ADC_REG_SetSequencerLength(ADC1, LL_ADC_REG_SEQ_SCAN_ENABLE_5RANKS);

ADC, set rank:
LL_ADC_REG_SetSequencerRanks(ADC1, LL_ADC_REG_RANK_1, LL_ADC_CHANNEL_0);
LL_ADC_REG_SetSequencerRanks(ADC1, LL_ADC_REG_RANK_2, LL_ADC_CHANNEL_1);
LL_ADC_REG_SetSequencerRanks(ADC1, LL_ADC_REG_RANK_3, LL_ADC_CHANNEL_2);
LL_ADC_REG_SetSequencerRanks(ADC1, LL_ADC_REG_RANK_4, LL_ADC_CHANNEL_3);
LL_ADC_REG_SetSequencerRanks(ADC1, LL_ADC_REG_RANK_5, LL_ADC_CHANNEL_4);

ADC, disable continuous mode:
LL_ADC_REG_SetContinuousMode(ADC1, LL_ADC_REG_CONV_SINGLE);

ADC, enable DMA request:
LL_ADC_REG_SetDMATransfer(ADC1, LL_ADC_REG_DMA_TRANSFER_LIMITED);

DMA, enable interrupt:
LL_DMA_EnableIT_TC(DMAx, Streamy);
NVIC_EnableIRQ(DMAx_Streamy_IRQn);

Start conversion:
LL_DMA_EnableStream(DMAx, Streamy);
LL_ADC_REG_StartConversion(ADC1);

ISR at end of conversion:
void DMAx_Streamy_IRQHandler(void)
{
    if (LL_DMA_IsActiveFlag_TC(DMAx, Streamy))
    {
        LL_DMA_ClearFlag_TC(DMAx, Streamy);

        // All 5 ADC values are now in buffer[]
        // Sequence finished
    }
}
